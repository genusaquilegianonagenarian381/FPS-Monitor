## 🔍 Troubleshooting Performance Data

Sometimes data might not show correctly. Try these tips if you face issues:

1. Close the software and restart it as an administrator. Right-click the shortcut and select Run as administrator.
2. Update your graphics card drivers. Old drivers often cause conflicts with performance monitoring tools.
3. Check the Settings menu to ensure the correct hardware sensors are selected.
4. If the overlay disappears, press the reset button in the Display preferences to return to default coordinates.

## 💡 Tips for Better Monitoring

*   Monitor GPU usage to see if your graphics card reaches 100 percent. This often causes stuttering.
*   Keep an eye on RAM usage. If usage stays near your total capacity, your computer might struggle to keep games smooth.
*   Check CPU temperatures. High heat can lead to a drop in performance as your processor lowers its speed to stay cool.
*   Use the data logs to compare performance before and after changing game graphics settings.
